# For organizations

Cat Dad Games is free to use, in full, by any organization that serves
older adults, people with disabilities, students, or the general public.
There is no licensing form, no usage fee, and no account to create.

This page is a practical how-to for staff at libraries, senior centers,
care institutions, classrooms, and other shared spaces.

## What you can do, without asking

- Bookmark this site on a public computer.
- Set it as the home page in a browser.
- Project it onto a screen for a group.
- Print this page, or any game's detail page, to share with residents,
  students, or family.
- Link to any page from your own site or newsletter.

## What is here, in plain terms

- A short, curated list of in-browser games. Each game opens in a new tab.
- No accounts to create. No data is collected about players.
- No ads, no upsells, no pop-ups, no third-party trackers.
- Every game lists its accessibility characteristics up front, so you can
  pick one that fits the person in front of you.

## Ideas for use

- **Common room or lobby computer.** Set this site as the home page so
  anyone can sit down and start playing.
- **One-on-one volunteer sessions.** Use the catalog to find a game that
  matches a participant's interests (puzzle, trivia, etc.) and their
  accessibility needs.
- **Group activities.** Look for games tagged "party" or "family" — many
  work on a single screen for a small group.
- **Computer classes.** The site itself is a good way to practice basic
  web skills: clicking, scrolling, opening and closing tabs.

## Tips that help older or newer users

- Show people the **text-size control** in the header. Bigger text is one
  click away.
- Use **Auto** mode for the theme — it follows the operating system, so the
  site looks consistent with whatever else is on the device.
- Remind people that **closing a game's tab brings them back to this site**.
  Nothing breaks if they close a tab by accident.
